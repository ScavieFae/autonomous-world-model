# LeWorldModel: Stable End-to-End JEPA from Pixels

**Paper:** arXiv 2603.19312 (March 2026)
**Authors:** Maes, Le Lidec, Scieur, LeCun, Balestriero (Mila, NYU, Samsung SAIL, Brown)
**Code:** https://github.com/lucas-maes/le-wm
**Companion (SIGReg):** arXiv 2511.08544 (LeJEPA, Balestriero & LeCun)

## Core Idea

JEPA-based latent world model that predicts dynamics in learned embedding space, not pixel space. Trains end-to-end from raw pixels with only two loss terms. No pretrained encoder, no EMA teacher, no stop-gradient.

## Architecture (~15M params)

- **Encoder:** ViT-tiny (5M params). Patch size 14, 12 layers, 3 heads, hidden dim 192. Raw pixels → CLS token → 1-layer MLP projector with BatchNorm → 192-dim embedding. One frame = one 192-dim vector.
- **Predictor:** Transformer (10M params). 6 layers, 16 heads, 10% dropout. Causal masking. History of N frame embeddings → next embedding prediction. Actions injected via **Adaptive Layer Normalization (AdaLN)** at each layer — zero-initialized so action conditioning ramps up gradually.

## Training

```
L = L_pred + λ * SIGReg(Z)
```

- `L_pred` = MSE between predicted next embedding and actual next embedding (teacher forcing)
- `SIGReg(Z)` = sketched isotropic Gaussian regularizer (Epps-Pulley test on random projections)
- `λ = 0.1` — the only hyperparameter to tune
- Fully offline, reward-free. End-to-end gradients. No EMA, no stop-gradient.

## SIGReg — The Key Innovation

The central problem in JEPA is representation collapse: if the encoder maps everything to the same point, prediction loss trivially goes to zero. Prior approaches use fragile heuristics (EMA teacher, multi-term VICReg with 7 loss terms).

SIGReg enforces that embeddings follow an isotropic Gaussian distribution via the Cramer-Wold theorem: project embeddings onto M random unit-norm directions, apply Epps-Pulley univariate normality test on each projection, aggregate. Convergence guarantee: SIGReg(Z) → 0 iff distribution converges weakly to N(0,I).

Replaces all heuristic anti-collapse machinery with one mathematically grounded term.

## Action Conditioning

Actions are continuous vectors, injected into the predictor via AdaLN (Adaptive Layer Normalization) at each transformer layer. AdaLN modulates LayerNorm parameters (scale and shift) based on action input. Zero-initialized so it ramps up gradually. Same mechanism as DiT (diffusion transformers).

NOT tokenized separately. Actions modulate dynamics prediction without being encoded as sequence tokens.

## Inference / Planning

Cross-Entropy Method (CEM) in latent space. Encode start and goal images. Sample candidate action sequences, roll out through predictor, pick those whose final embedding is closest to goal. MPC: execute first K actions, replan. Full planning in ~1 second (48x faster than DINO-WM due to single 192-dim token vs ~200 patch tokens).

## Key Results

- Outperforms PLDM (other end-to-end method) on all challenging tasks. +18% success on Push-T.
- Competitive with DINO-WM (which uses pretrained DINOv2 on 124M images). Beats DINO-WM on Push-T even when DINO-WM uses proprioceptive inputs.
- Linear probes recover physical quantities (agent location, block location, angles) from embeddings.
- **Emergent temporal path straightening:** latent trajectories become increasingly straight over training with no explicit regularization.
- **Violation-of-expectation tests:** correctly assigns high surprise to physically impossible events (p<0.01).

## Limitations

- Short planning horizons only. Hierarchical world modeling needed for longer horizons.
- Needs offline datasets with sufficient interaction coverage.
- SIGReg may underperform in very low-complexity environments (Two-Room failure case).
- Depends on action labels — inverse dynamics could remove this.

## Relevance to AWM/Melee

1. Operates in continuous latent space, not discrete tokens — different paradigm from our Mamba2 approach
2. The 15M param / single GPU / few hours profile is interesting for what's achievable at small scale
3. AdaLN action conditioning is a clean alternative to our input concatenation approach
4. Temporal path straightening is a potential diagnostic for our rollouts
5. The core question: can we adapt latent-space JEPA prediction to structured game state?
